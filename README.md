# Con-Quest - Comic-Con Scavenger Hunt PWA

A Progressive Web App that gamifies the Comic-Con convention experience through interactive scavenger hunts. Complete tasks, earn points, and unlock rewards while exploring the event.

## Features

- **Offline-First**: Works without internet using Service Workers
- **Task System**: Complete challenges around the convention
- **Points & Rewards**: Gamification system to track progress
- **Responsive Design**: Works on all devices
- **PWA**: Installable on mobile devices

## Tech Stack

- **JavaScript** - Core functionality
- **Node.js** - Backend services
- **Service Workers** - Offline capabilities
- **PWA** - Progressive Web App features
- **Figma** - UI/UX design
